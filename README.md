
# 🍔 Zesto — AI-Powered Food Delivery Platform

Zesto is a full-stack food delivery application built using the MERN stack.
It integrates an intelligent AI agent that helps users make better food choices through personalized recommendations, healthy meal suggestions, and real-time conversational assistance.

## 🧠 How It Works


Zesto combines traditional food delivery features with an AI-powered recommendation system.

🤖 AI Agent Capabilities
Provides personalized food suggestions
Recommends healthy meal combinations
Answers user queries about food and nutrition
Helps users make smarter and balanced choices
## 🏗️ Architecture

🔹 Frontend
- Built using React.js
- Responsive UI for browsing, ordering, and interacting with AI

🔹 Backend
- Node.js + Express.js
- RESTful APIs for user, orders, and menu management

🔹 Database
- MongoDB for storing users, orders, and food items

🔹 AI Layer
- Integrated LLM (via Groq) for recommendations and conversations
## ✨ Features

- 🍔 Browse and order food online
- 🤖 AI-powered food recommendation system
- 🥗 Healthy combo meal suggestions
- 💬 Conversational AI assistant
- 🔐 User authentication and authorization
- 📦 Order management system
